# Field Types

Blockstudio provides a variety of field types for building block interfaces.

## General Properties

All fields share these common properties:

| Property | Type | Description |
|----------|------|-------------|
| `id` | string | Unique identifier |
| `type` | string | Field type |
| `label` | string | Display label |
| `default` | mixed | Default value |
| `help` | string | Help text |
| `position` | string | `inspector` or `toolbar` |
| `conditions` | array | Conditional logic |

---

## Text

A single-line text input field.

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "title",
        "type": "text",
        "label": "Title"
      }
    ]
  }
}
```

| Property | Type | Description |
|----------|------|-------------|
| `multiple` | boolean | Allow multiple files |

## Group

A collapsible group of fields.

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "type": "group",
        "title": "Settings",
        "initialOpen": true,
        "attributes": [
          {
            "id": "title",
            "type": "text",
            "label": "Title"
          },
          {
            "id": "count",
            "type": "number",
            "label": "Count"
          }
        ]
      }
    ]
  }
}
```

### With ID Prefix

When a group has an `id`, all inner attribute IDs are prefixed:

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "settings",
        "type": "group",
        "title": "Settings",
        "attributes": [
          {
            "id": "title",
            "type": "text",
            "label": "Title"
          }
        ]
      }
    ]
  }
}
```

Access via `$a['settings_title']` or use the helper:

```php title="index.php"
<?php
$settings = bs_get_group($a, 'settings');
echo $settings['title'];
?>
```

## Repeater

A repeatable group of fields.

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "items",
        "type": "repeater",
        "label": "Items",
        "min": 1,
        "max": 10,
        "textButton": "Add Item",
        "attributes": [
          {
            "id": "title",
            "type": "text",
            "label": "Title"
          },
          {
            "id": "description",
            "type": "textarea",
            "label": "Description"
          }
        ]
      }
    ]
  }
}
```

### Rendering

```php title="index.php"
<?php foreach ($a['items'] as $item): ?>
  <div class="item">
    <h3><?php echo $item['title']; ?></h3>
    <p><?php echo $item['description']; ?></p>
  </div>
<?php endforeach; ?>
```

```twig title="index.twig"
{% for item in a.items %}
  <div class="item">
    <h3>{{ item.title }}</h3>
    <p>{{ item.description }}</p>
  </div>
{% endfor %}
```

### textMinimized

Customize the minimized label:

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "items",
        "type": "repeater",
        "label": "Items",
        "textMinimized": {
          "id": "title",
          "fallback": "Untitled",
          "prefix": "Item {index}: "
        },
        "attributes": [...]
      }
    ]
  }
}
```
