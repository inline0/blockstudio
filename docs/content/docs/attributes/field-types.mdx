# Field Types

Blockstudio provides a variety of field types for building block interfaces.

## General Properties

All fields share these common properties:

| Property | Type | Description |
|----------|------|-------------|
| `id` | string | Unique identifier for the attribute |
| `type` | string | The field type |
| `label` | string | Display label shown in the editor |
| `default` | mixed | Default value for the field |
| `help` | string | Help text displayed below the field |
| `position` | string | Where to display: `inspector` (default) or `toolbar` |
| `conditions` | array | Conditional logic rules |
| `hidden` | boolean | Hide the field from the UI |
| `width` | string | Field width: `25`, `33`, `50`, `66`, `75`, or `100` |
| `returnFormat` | string | Return format: `value`, `label`, or `both` |

## Text

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "title",
        "type": "text",
        "label": "Title"
      }
    ]
  }
}
```

| Property | Type | Description |

| `placeholder` | string | Placeholder text |
| `maxLength` | number | Maximum character length |

## Files

Media/file upload field.

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "image",
        "type": "files",
        "label": "Image",
        "multiple": false
      }
    ]
  }
}
```

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `multiple` | boolean | Allow multiple files |
| `allowedTypes` | array | Allowed mime types (e.g., `["image"]`) |
| `size` | boolean | Show image size dropdown |

### Return Value

Returns an object (or array if `multiple: true`) with:

| Property | Type | Description |
|----------|------|-------------|
| `ID` | number | Attachment ID |
| `url` | string | File URL (selected size if using `size`) |
| `alt` | string | Alt text |
| `title` | string | File title |
| `caption` | string | File caption |
| `width` | number | Image width |
| `height` | number | Image height |
| `mime` | string | Mime type |
| `sizes` | object | Available image sizes |

### Placeholders

- `{block.name}` - Block name

## Group

A collapsible group of fields.

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "type": "group",
        "title": "Settings",
        "initialOpen": true,
        "attributes": [
          {
            "id": "title",
            "type": "text",
            "label": "Title"
          },
          {
            "id": "count",
            "type": "number",
            "label": "Count"
          }
        ]
      }
    ]
  }
}
```

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `id` | string | Optional ID to prefix inner attributes |
| `title` | string | Group title |
| `initialOpen` | boolean | Whether group is open by default |
| `attributes` | array | Nested attributes |

### With ID Prefix

When a group has an `id`, all inner attribute IDs are prefixed:

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "settings",
        "type": "group",
        "title": "Settings",
        "attributes": [
          {
            "id": "title",
            "type": "text",
            "label": "Title"
          }
        ]
      }
    ]
  }
}
```

Access via `$a['settings_title']` or use the helper:

```php title="index.php"
<?php
// bs_get_group($attributes, $groupName)
$settings = bs_get_group($a, 'settings');
echo $settings['title'];
?>
```

## Repeater

A repeatable group of fields.

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "items",
        "type": "repeater",
        "label": "Items",
        "min": 1,
        "max": 10,
        "textButton": "Add Item",
        "attributes": [
          {
            "id": "title",
            "type": "text",
            "label": "Title"
          },
          {
            "id": "description",
            "type": "textarea",
            "label": "Description"
          }

      }
    ]

```

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `min` | number | Minimum number of items |
| `max` | number | Maximum number of items |
| `textButton` | string | Add button text |
| `textMinimized` | string \| object | Minimized row label |

### textMinimized

Customize the minimized row label:

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "items",
        "type": "repeater",
        "label": "Items",
        "textMinimized": {
          "id": "title",
          "fallback": "Untitled",
          "prefix": "Item {index}: ",
          "suffix": " - end"
        },
        "attributes": [
          {
            "id": "title",
            "type": "text",
            "label": "Title"
          }
        ]
      }
    ]
  }
}
```

| Property | Type | Description |
|----------|------|-------------|
| `id` | string | Attribute ID to display |
| `fallback` | string | Fallback text if empty |
| `prefix` | string | Text before value |
| `suffix` | string | Text after value |

Use `{index}` placeholder for the row number.
