# Tailwind CSS

Blockstudio includes built-in Tailwind CSS support for the block editor.

## Enabling Tailwind

Enable Tailwind in your `blockstudio.json`:

```json title="blockstudio.json"
{
  "tailwind": {
    "enabled": true
  }
}
```

## Classes Field

Use the `classes` field type with Tailwind autocomplete:

```json title="block.json"
{
  "blockstudio": {
    "attributes": [
      {
        "id": "classes",
        "type": "classes",
        "label": "CSS Classes",
        "tailwind": true
      }
    ]
  }
}
```

## Custom Configuration

Add custom Tailwind configuration using CSS syntax (Tailwind v4):

```json title="blockstudio.json"
{
  "tailwind": {
    "enabled": true,
    "config": "@theme { --color-brand: #ff6b6b; }"
  }
}
```

## Custom Classes

Define reusable custom classes:

```json title="blockstudio.json"
{
  "tailwind": {
    "enabled": true,
    "customClasses": [
      {
        "className": "btn-primary",
        "value": "bg-blue-500 text-white px-4 py-2 rounded"
      },
      {
        "className": "card",
        "value": "bg-white shadow-lg rounded-lg p-6"
      }
    ]
  }
}
```

Custom classes appear in the autocomplete suggestions and are compiled using `@apply`.

## Editor Preview

Tailwind classes are compiled in real-time within the block editor, giving you instant visual feedback as you add classes.
